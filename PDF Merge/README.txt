Required:
- Python installation (overwrite system variables)
- PIP installation (Run CMD in install folder to install pip)

Modules to install over PIP:  (im CMD: py -m pip install PyPDF2)
- PyPDF2

How it works:
Put your PDF-Files into the "PDFs"-Folder and run the "Merge.bat" script.
Please make sure you don't use subfolders.
The sort is the default explorer sort by filenames.
