THIS WORKS ONLY FOR ACADSHX TEXT-OBJECTS!

Modules to install over PIP:  (im CMD: py -m pip install PyPDF2)
- PyPDF2
- reportlab

Python Version: 3.9.6

Start it over CMD:
<path to python.exe (3.9.6)> <path to SHXtoText.py> <path to your PDF>

Example:
"C:\Python\Python396\python.exe" "C:\Python\Python396\SHXtoText.py" "C:\ExamplePDF.pdf"
